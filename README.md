# dotfiles

Personal tooling, automation scripts, and system configuration.

## Contents

| Folder | What's in it |
|--------|-------------|
| [`/ahk`](./ahk) | AutoHotkey v2 scripts for Windows automation |

## Setup

Clone the repo to a stable local path:

```
git clone https://github.com/<your-username>/dotfiles C:\Users\<your-username>\dotfiles
```

Each folder has its own README with specific setup instructions.
